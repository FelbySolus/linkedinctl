# LinkedIn Launch Pack (3-4 Prompt Format)

Use these prompts in sequence to publish `v0.0.1` clearly with proof.

## Prompt 1 - Release Post Draft

"Write a concise LinkedIn post announcing `linkedinctl v0.0.1`, an open-source CLI for strict JSON-driven LinkedIn profile automation. Tone: technical builder. Include: strict schema validation, headless Patchwright adapter, cover renderer with safe-zone text placement, and fail-closed pipeline controls. End with a CTA to the GitHub repo."

## Prompt 2 - Proof Section

"Add a short 'Proof' section to the post with 4 bullets based on these facts: TypeScript build passed, 8 Python tests passed, plan/apply pipeline succeeded with 4 operations, cover renderer generated 1584x396 output with deterministic coordinates. Keep it credible and non-hype."

## Prompt 3 - Carousel/Media Captions

"Create 4 slide captions for a LinkedIn carousel:
1) Problem and constraints,
2) Architecture and strict pipeline,
3) CLI commands and result snippets,
4) v0.0.1 release + GitHub link.
Each caption max 220 characters."

## Prompt 4 - Comment/Follow-up

"Draft one follow-up comment for my own post inviting collaborators to test edge cases (UI flakiness, selector variants, localization differences) and open issues/PRs."
